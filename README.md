# 📩 Spam Message Classifier (Flask + ML)

This is a simple machine learning web app that classifies SMS/Email messages as **Spam** or **Not Spam** using Flask.

---

## 🚀 Features

- Text preprocessing using NLTK
- TF-IDF Vectorization
- Machine Learning model prediction
- Clean UI with HTML + CSS
- Deployed using Render

---

## 🛠 Tech Stack

- Python
- Flask
- Scikit-learn
- NLTK
- HTML + CSS
